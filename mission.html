Mission & Roadmap

What we aim to achieve in the short, medium and long term — and the phased path to get there.

Short term

An executable preparedness process

Establish a working cross-national multi-database research preparedness process — collaboration network, database inventory, CDM conversion, common protocols, analysis modules, visualization and QA / QC workflows.

Medium term

Reusable RWE infrastructure

Develop reusable cross-national RWE research infrastructure — an analysis-module library, a topic-appraisal process, distributed-analysis SOPs, result-integration templates, dashboards and a shared interpretation framework.

Long term

An internationally connected platform

Build an Asia-to-international multi-database study preparedness platform, so cross-national real-world data can support disease burden, treatment effect, drug safety, clinical, regulatory, policy, HTA and industry decisions — in time.

Development phases

From foundation to timely evaluation

Phase 1
Foundation

Build the collaboration team, conduct database inventory, set up research-governance structures and begin initial CDM conversion.

Phase 2
Standardization

Establish a common data model, common study protocols, shared codelists, analysis modules and data-quality check processes.

Phase 3
Integration

Build cross-national data-structure mapping, integrated analysis workflows, a topic-appraisal mechanism and a result-integration framework.

Phase 4
Visualization

Build database readiness dashboards, results visualization, cross-country comparison charts, heterogeneity views and decision dashboards.

Phase 5
Timely Evaluation

Rapidly run multi-country studies on disease and treatment questions, integrate results, interpret evidence and support clinical, policy, regulatory and industry decisions.
